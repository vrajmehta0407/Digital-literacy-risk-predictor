# Elderly UI Design Guidelines

## Philosophy
The goal is to reduce cognitive load and remove the fear of making mistakes.

## 1. Visual Design
*   **Contrast**: Use high contrast ratios (e.g., Black text on White background).
*   **Color coding**:
    *   **Green (#4CAF50)**: Safe / Good / Go.
    *   **Amber (#FFC107)**: Caution / Wait / Check.
    *   **Red (#F44336)**: Danger / Stop / Alert.
*   **Typography**:
    *   Minimum font size: 24sp.
    *   Sans-serif fonts only.
    *   Avoid italics.

## 2. Interaction
*   **Touch Targets**: Minimum 60dp x 60dp for all buttons.
*   **Navigation**: No hamburger menus, no tabs, no scrolling if possible.
*   **Gestures**: Single tap only. No swipes, no pinch-to-zoom.

## 3. Communication
*   **Language**: Simple, direct, non-technical.
    *   *Bad*: "Malicious payload detected in SMS PDU."
    *   *Good*: "Warning. This message is unsafe."
*   **Feedback**: Always provide audio feedback for important states.
